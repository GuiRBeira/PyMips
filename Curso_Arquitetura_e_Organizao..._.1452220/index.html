Ementa:
Arquiteturas gerais de Computadores. Aritmética para computadores com inteiros e ponto flutuante. Reduced Instruction Set Computer(RISC). Complex Instruction Set Computer(CISC).Unidade de Processamento de Dados (CPU). Arithmetic Logic Unit(ALU). Instruções e linguagem de máquina. Modos de endereçamento. Hierarquia de memória. Pipeline. Mecanismos de interrupção. Interface com periféricos. Arquiteturas Paralelas e não Convencionais.
Avaliação:

Serão realizadas avaliações práticas no ambiente virtual moodle Valor 0 - 10. As médias das avaliações corresponderá a 60 por cento da nota total. Dos 40 por cento restantes, 30 serão referentes a participação do aluno no desenvolvimento de um projeto prático, ao qual, será atribuída uma nota entre 0 - 10. Os 10 por cento restantes serão atribuídos por apresentação de trabalhos orais (seminários).


A nota final será dada pela fórmula:



NF = média_avaliações*0,6 + Nota_projeto*0,3 + seminários*0,1.
