1) Implemente o seguinte programa em assembly do mips:

           int somador(int n){

                      if(n > 0)
                           return (n + somador(n-1));
                     else
                           return 0;
           }

           void main(){

              int n;
              scanf("%d", &n);
              printf("Resultado  %d\n\n",somador(n));
           }

Dica: Siga as diretivas do mips para as chamadas de procedimentos e funções.


2) Implemente uma versão da função strlen(str) em assembly do mips utilizando ponteiro, e outra versão utilizando array. Compare as duas versões.

3) Implemente uma versão da função strcmp(str1, str2) em assembly do mips.
