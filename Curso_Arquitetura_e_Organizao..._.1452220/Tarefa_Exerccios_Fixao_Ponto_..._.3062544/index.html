Descrição

Traduza os números a seguir para binário, de acordo com o formato de números com ponto flutuante apresentados em aula.

os resultados devem apresentar o seguinte formato de 32 bits:

Exemplo:

Sinal       Expoente               Mantissa

  0           00011101  10101111000000000000000

A partir disso calcule:

a) 37,8

b) 90,56

c) 14,1

d) 19,99
